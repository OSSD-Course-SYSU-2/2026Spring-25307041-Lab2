# 迷宫游戏测试指南

## 测试环境准备

1. 确保DevEco Studio已正确安装
2. 确保HarmonyOS SDK已配置
3. 创建或启动DevEco虚拟机

#### 1.1 应用启动测试
- [ ] 应用能够正常启动

#### 1.2 迷宫生成测试

## 测试方法

### 手动测试步骤

1. **启动应用**
   - 在DevEco Studio中点击运行
   - 选择虚拟机设备
   - 等待应用安装并启动

2. **测试移动功能**
   - 观察初始迷宫布局
   - 依次点击↑↓←→按钮
   - 验证角色移动是否正确

3. **测试炸毁功能**
   - 移动角色面向可炸毁方块
   - 点击💥按钮
   - 验证方块是否被炸毁

4. **测试胜利条件**
   - 通过移动和炸毁到达出口
   - 验证胜利提示是否显示

5. **测试重置功能**
   - 点击"重新开始"按钮
   - 验证游戏是否正确重置
